# File Share App

A lightweight local-network file sharing app for Windows laptop and Android mobile.

## Features
- Upload images and videos from the browser
- Download files from the same local network
- Works on the same Wi-Fi network

## Run

```bash
npm install
npm start
```

Then open:

```text
http://192.168.2.106:3000
```

On Android, use the same Wi-Fi network and open the same address in Chrome.

## Notes
- Files are stored in the `uploads` folder.
- Only image and video files are accepted.
